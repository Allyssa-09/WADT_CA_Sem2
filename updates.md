# Travel Bucket List - Weekly Updates

## Explore Hub:  Project Overview

To give a bit more context , Explore Hub is a travel bucket list web app where multiple users can create an account and save destinations they want to visit or have already visited.

The reason for two tables is:
- A destinations table — to store all available travel destinations, including name, country, continent, category, image, and other details
- A bucket_list table — to store which destinations each user has saved to their personal list, along with their own status (visited/wishlist), notes, rating, and dates

The two tables are linked through dest_id, so each entry in the bucket_list points to a destination in the destinations table.
